# am-editor

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

Start the dev server by SSR

```bash
$ yarn ssr
```

Start the ot server

```bash
$ cd ot-server
$ yarn start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
