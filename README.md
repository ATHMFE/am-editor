# am-editor

暂时处于测试阶段，还未发布任何包

编辑器引擎纯 JavaScript 编写。插件可配合 React 或 Vue 等主流前端库渲染

## Getting Started

Install dependencies,

```bash
$ yarn
```

Start the dev server,

```bash
$ yarn start
```

Start the dev server by SSR

```bash
$ yarn ssr
```

Start the ot server

```bash
$ cd ot-server
$ yarn start
```

Build documentation,

```bash
$ npm run docs:build
```

Build library via `father-build`,

```bash
$ npm run build
```
